----In this chapter we are working on conditionals & loops in python. Understand how conditionals & loops works in python.----

1. if Statements: There can be zero or more elif parts, and the else part is optional. The keyword ‘elif’ is short for ‘else if’, and is useful to avoid excessive indentation. An if … elif … elif … sequence is a substitute for the switch or case statements found in other languages.

2. for statement: Python’s for statement iterates over the items of any sequence (a list or a string), in the order that they appear in the sequence.

Enumerate in for loop: create the items with their index value

Zip function: zip(*iterables, strict=False)
Iterate over several iterables in parallel, producing tuples with an item from each one.

Range function: If you do need to iterate over a sequence of numbers, the built-in function range() comes in handy. It generates arithmetic progressions

3. While Statement: The while statement is used for repeated execution as long as an expression is true

4. Break Statement: A break statement executed in the first suite terminates the loop without executing the else clause’s suite

5. Continue Statement:  A continue statement executed in the first suite skips the rest of the suite and goes back to testing the expression.
